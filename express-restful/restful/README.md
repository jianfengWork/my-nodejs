### RESTFUL
```
  restful  从 URL 的格式来表述的
  get      http://localhost:3030/toEditBook/:id
  delete   http://localhost:3030/deleteBook/:id

  传统的 URL 风格
  get      http://localhost:3030/toEditBook?id=1
  delete   http://localhost:3030/deleteBook?id=1

  cd express-restful
  node server.js
  http://localhost:3030/toEditBook/1
```
